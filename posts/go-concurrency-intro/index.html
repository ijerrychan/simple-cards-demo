Go语言并发编程入门 #

Go语言以其内置的并发特性而闻名，通过goroutine和channel，开发者可以轻松编写高效的并发程序。

Goroutine #

Goroutine是Go语言中轻量级的线程。它们由Go运行时管理，而不是操作系统。启动一个goroutine非常简单，只需在函数调用前加上go关键字即可。

package main

import (
	"fmt"
	"time"
)

func sayHello() {
	time.Sleep(1 * time.Second)
	fmt.Println("Hello from goroutine!")
}

func main() {
	go sayHello()
	fmt.Println("Hello from main!")
	time.Sleep(2 * time.Second) // 等待goroutine完成
}

Channel #

Channel是goroutine之间通信的管道。它们允许goroutine安全地发送和接收数据。

package main

import "fmt"

func sum(s []int, c chan int) {
	sum := 0
	for _, v := range s {
		sum += v
	}
	c <- sum // 将和发送到channel
}

func main() {
	s := []int{7, 2, 8, -9, 4, 0}

	c := make(chan int)
	go sum(s[:len(s)/2], c)
	go sum(s[len(s)/2:], c)

	x, y := <-c, <-c // 从channel接收
	fmt.Println(x, y, x+y)
}

通过goroutine和channel的组合，Go语言提供了一种强大而简洁的并发编程模型。
